This project (w.i.p.) implements and compares methods for filtering in State-Space models, starting from the linear Gaussian case up to nonlinear non-Gaussian high-dimensional scenarios, including Stochastic Volatility models.
It covers the Kalman Filter and its variants (EKF/UKF), particle filters, particle flow methods (EDH/LEDH, invertible PF-PF), kernel-embedded particle flows, 
differentiable particle filtering with entropy-regularized optimal transport (Sinkhorn) and neural optimal transport.
